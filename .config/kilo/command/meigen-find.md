# 工作流目标

用于在明确的视觉探索场景下快速查找参考图、风格方向与提示词线索。

本 command 属于长尾试点能力，只负责“找灵感”，不负责图片生成结果质量，也不负责把视觉参考转成前端代码。

## Workflow

### 步骤 1：确认检索意图
- 接收用户给出的关键词、主题、风格或对象描述
- 如未提供关键词，则按“浏览灵感”处理

### 步骤 2：调用画廊搜索
- 优先调用 MeiGen gallery 搜索能力获取参考结果
- 结果数量保持紧凑，避免把 command 变成完整浏览系统

### 步骤 3：整理返回结果
- 输出编号、简短提示词摘要、分类与可见参考图
- 保持结果可扫描，避免长篇说明

### 步骤 4：声明边界
- 明确该 command 只负责灵感检索
- 如用户后续需要图片生成或前端实现，应作为独立任务处理

### 步骤 5：处理失败与降级
- 如果 MeiGen MCP 未配置或不可用，明确说明前置依赖缺失
- 不尝试在本 command 内自动扩展为别的设计工作流
